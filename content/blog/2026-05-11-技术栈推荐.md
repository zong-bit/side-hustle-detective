# 程序员接单需要会什么技术？

## 热门需求（能马上接到单的）
- Web开发：React、Vue、Node.js
- 小程序开发：微信小程序
- AI集成：OpenAI API、DeepSeek API
- WordPress：建站需求大

## 加分技能（单价更高的）
- AI Agent开发
- RAG知识库搭建
- 自动化工作流（n8n/Zapier）
- 云服务部署（Vercel/AWS）

## 学习路线
1. 先学一门核心技能（推荐React + Node.js）
2. 再学AI集成（调用API就行）
3. 最后学部署（Vercel免费够用）

## 推荐
已有技术栈：React + Node.js + AI API
完全够接单。
