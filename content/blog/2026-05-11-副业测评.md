## 新手最佳上手路径

**第1周**：注册闲鱼，发布第一个服务商品
**第2-3周**：在小红书发布第一篇笔记
**第4-6周**：在知乎回答10个相关问题
**第7-8周**：评估效果，选定1-2个重点平台
**第9-12周**：深耕选定的平台，建立稳定收入

不要贪多，3个月内只专注2-3个平台足矣。

## 多平台策略建议

### 内容分发策略
一篇内容在多个平台同时发布：
- 写一篇文章 → 发知乎 + 发小红书文案 + 发公众号
- 做一个视频 → 发B站 + 发抖音 + 发微信视频号
- 做一张图 → 发小红书 + 发闲鱼商品图 + 发朋友圈

### 平台联动策略
- 知乎回答问题 → 引导关注公众号
- 小红书发笔记 → 引导去闲鱼下单
- 闲鱼服务客户 → 推荐加入知识星球
- B站视频 → 引导加私域

## 平台选择决策树

```
你擅长写作吗？
├── 是 → 你想做深度还是短平快？
│   ├── 深度 → 知乎、公众号、B站
│   └── 短平快 → 小红书、抖音
└── 不擅长写作
    └── 你有技术技能吗？
        ├── 有 → 程序员客栈、Upwork
        └── 没有
            └── 闲鱼、贴吧、线下服务
```
