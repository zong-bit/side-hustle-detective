## 热门需求（能马上接到单的）
- Web开发：React、Vue、Node.js
- 小程序开发：微信小程序
- AI集成：OpenAI API、DeepSeek API
- WordPress：建站需求大

## 加分技能（单价更高的）
- AI Agent开发
- RAG知识库搭建
- 自动化工作流（n8n/Zapier）
- 云服务部署（Vercel/AWS）

## 学习路线
1. 先学一门核心技能（推荐React + Node.js）
2. 再学AI集成（调用API就行）
3. 最后学部署（Vercel免费够用）

## 推荐
已有技术栈：React + Node.js + AI API
完全够接单。

<!-- EXPANDED: 技术栈详解 -->
React + Node.js 是目前接单量最大的技术组合。React 负责前端页面，Node.js 负责后端接口，AI API 负责智能功能。这套组合能接的活包括：企业官网、小程序、管理后台、数据看板、AI聊天机器人、内容管理系统等。学习资源推荐：React 官方文档（react.dev）、Node.js 官方文档、B站上的免费教程。重点掌握：组件化开发、RESTful API、数据库操作（MongoDB 或 MySQL）、第三方 API 调用。

<!-- EXPANDED: 接单平台推荐 -->
国内接单平台：1）猪八戒——项目多但竞争激烈，适合新手练手；2）程序员客栈——技术门槛较高，单价也高；3）码市——开源社区背景，项目质量不错；4）闲鱼——发布服务描述，适合小单；5）小红书——发技术文章引流，客户主动找你。国外平台：1）Upwork——全球最大自由职业平台，单价高但需要英语；2）Fiverr——适合做标准化服务（比如"¥200帮你搭建一个网站"）；3）Freelancer.com——项目多但质量参差不齐。建议先从国内平台开始，有了一定经验后再接海外单。

<!-- EXPANDED: 新手接单避坑指南 -->
新手接单最容易踩的坑：1）报价太低——不要为了接单故意压低价格，合理的报价是对自己的尊重。一个简单网站 ¥500-1500，复杂项目 ¥2000-5000。2）需求不明确就开工——开工前一定要和客户确认需求文档，包括功能列表、设计风格、交付标准。3）没有定金就干活——至少收 30%-50% 定金。4）没有验收标准——和客户确认好验收标准，避免反复修改。5）不签合同——简单的项目可以用文字记录确认，复杂项目建议签正式合同。

<!-- EXPANDED: 从0到1的实操路径 -->
第一个月：学 React + Node.js 基础，做 2-3 个小项目练手（比如个人博客、待办事项App）。第二个月：学 AI API 集成，做一个 AI 聊天应用。第三个月：在猪八戒/闲鱼发布服务，开始接小单。第四个月：根据前几个单子的反馈优化服务描述和报价，尝试更高的单价。关键：不要等"学完了"再接单——在接单中学是最快的成长方式。每接一个单就总结一次经验，慢慢你的技术、报价、沟通能力都会提升。

<!-- EXPANDED: 技术栈的替代方案 -->
React + Node.js 不是唯一选择。如果你更熟悉 Vue，Vue + Express 同样可以接单，而且国内中小客户对 Vue 的接受度更高。如果你不想学前端，纯后端方向（Python + FastAPI / Go）也可以接 API 开发、爬虫、自动化脚本等单。WordPress 方向门槛最低，不需要写代码，用现成主题和插件就能搭建网站，适合完全没有技术基础的程序员转型。不管选哪个技术栈，核心能力是一样的：需求分析、代码实现、沟通交付。

<!-- EXPANDED: 技术能力之外的软实力 -->
程序员做副业，技术只是基础，软实力同样重要：

1）沟通能力——能听懂客户需求，用客户听得懂的语言解释技术方案
2）时间管理——按时交付比完美交付更重要，客户宁愿要一个及时交付的 80 分作品，也不要一个迟到一周的 100 分作品
3）学习能力——技术更新很快，需要持续学习新工具和新框架
4）商业思维——理解客户为什么需要这个功能，而不仅仅是怎么做

这些软实力决定了你能走多远。
