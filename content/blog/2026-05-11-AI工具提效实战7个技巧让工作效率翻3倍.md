## 场景一：用ChatGPT写邮件、周报，告别"憋字"痛苦

### 痛点
每天花1小时写邮件和周报，尤其是英文邮件，来回斟酌措辞，时间就这么没了。

### 具体操作步骤

**1. 写周报（5分钟搞定）**

给AI一个简单的模板指令：

```
你是一位职场周报写作助手。请根据以下信息，帮我写一份周报：
- 本周工作：完成了用户增长方案初稿、分析了Q2数据、参加了3场跨部门会议
- 下周计划：提交终稿、和设计团队对齐视觉效果
- 本周成果：用户增长方案获主管认可
- 格式要求：按「本周重点→下周计划→遇到问题」三段式

语言：简洁干练，适合发在工作群里
```

**2. 写英文邮件（再也不怕语法错误）**

```
请帮我写一封英文邮件：
- 收件人：美国客户 Tom
- 目的：确认下周二的会议时间，并提供腾讯会议链接
- 语气：专业但友好
- 附件说明：会上会展示最新版产品原型

请提供正式版和简洁版两个版本。
```

> 💡 **小技巧**：把你常用的邮件模板保存为"自定义GPT"或提示词模板，下次直接复用。

## 场景二：AI做PPT，15分钟从零到完整演示文稿

### 痛点
每次做PPT都从找模板开始，排版、配色、字体……半天过去了，内容还没写多少。

### 具体操作步骤

**工具推荐：** Gamma、Beautiful.ai、讯飞智文、AIPPT

**以 Gamma 为例：**

1. 打开 Gamma.app，选择"用AI生成"
2. 输入主题描述，例如："2026年Q1市场运营复盘报告，包含粉丝增长、转化率、ROI分析、下季度策略"
3. AI自动生成大纲 → 你可以拖拽调整顺序
4. 选择视觉风格 → 一键生成完整PPT
5. 微调细节（改文案、换图）
6. 导出或在线分享

> 实测：原本4小时的PPT工作，压缩到15-20分钟。省下来的时间，用来打磨核心内容。

## 场景三：AI数据分析，Excel小白秒变分析师

### 痛点
数据堆在那儿，不会用透视表，不会写公式，更不会用Python做分析。

### 具体操作步骤

**工具推荐：** ChatGPT（Code Interpreter/Advanced Data Analysis）、Julius AI、网易有数

**用 ChatGPT 分析CSV数据：**

1. 把你的Excel/CSV文件直接上传到ChatGPT
2. 用自然语言提问，比如：
   - "帮我分析这个月各渠道的转化率排名"
   - "找出销售额最高的前10个SKU"
   - "看看用户流失率和哪些因素相关"
3. AI自动写Python代码做分析、生成图表
4. 告诉AI"用中文总结结论，用项目经理能看懂的语言"
5. 导出分析报告

**进阶用法：** 让AI预测趋势
```
根据过去12个月的销售数据，预测下3个月的销量趋势，
用时间序列模型，输出可视化图表和文字结论。
```

> 💡 **小技巧**：不用一次喂太多数据。先上传关键字段，跑通流程后，再逐步加数据。

## 场景四：AI会议纪要——再也不用边开会边疯狂打字

### 痛点
开会开了2小时，记了3页笔记，回头一看——根本看不懂自己写了什么。

### 具体操作步骤

**工具推荐：** 飞书妙记、通义听悟、Otter.ai、Fireflies.ai

**以飞书妙记为例：**

1. 会议开始前，开启飞书妙记（支持语言转文字）
2. AI实时转写，自动区分发言人
3. 会议结束后，AI自动生成：
   - ✅ 完整文字稿（可搜索）
   - ✅ 会议摘要（3句话概括）
   - ✅ 待办事项（自动提取Action Items）
4. 一键发送到群里，所有人同步

**进阶玩法：**
```
帮我从今天的会议记录中提取：
1. 所有需要我后续跟进的事项
2. 涉及「预算」的讨论结论
3. 有争议未达成一致的议题及各方观点
```

> 省下来的不是记录时间，而是**会后对齐信息**的大量沟通成本。

## 场景五：AI修图/设计，不懂PS也能出专业效果

### 痛点
想做个海报、修个产品图，找人设计太贵，自己学PS太慢。

### 具体操作步骤

**工具推荐：** Canva AI、Adobe Firefly、Remove.bg、美图Wink

**以 Canva AI 为例：**

1. 打开 Canva，选择"Magic Design"
2. 上传一张图片或输入主题描述
3. AI自动生成多种风格的设计方案
4. 使用"Magic Eraser"一键去掉背景杂物
5. 用"Magic Expand"扩展图片边界
6. 自动抠图、换背景、调整配色——全拖拽完成

**产品图批量处理流程：**
```
需求：一组50张产品实拍图，需要：
1. 自动抠图换白背景
2. 统一尺寸为 800×800
3. 批量调亮曝光不足的图片

建议工具：Remove.bg（抠图）+ Canva（批处理）
```

> 电商从业者尤其推荐——AI批量处理产品图，效率提升10倍不止。

## 场景六：用AI做竞品调研，一上午搞定一周的工作量

### 痛点
手动搜竞品、看官网、看评论、整理对比表——一周的活。

### 具体操作步骤

**工具推荐：** ChatGPT + Perplexity + 浏览器

```
你是一位市场研究员。请帮我完成以下竞品调研任务：

目标行业：在线英语教育
调研范围：国内Top 5竞品

请帮我整理：
1. 各竞品的主要产品形态（AI课/真人直播/录播）
2. 定价策略对比（客单价、付费模式）
3. 核心差异化卖点
4. 用户评价中出现的Top 5槽点

输出格式：
- 先用表格做概览对比
- 然后逐条分析
- 最后给出我的产品可以切入的机会点
```

**进阶操作：** 让AI帮你爬取和分析评论

把竞品在应用商店、知乎、小红书的用户评论截图或粘贴给AI，让它做情感分析和关键词提取——这是人工一周都干不完的活。

## 场景七：AI个人助理，管理你的时间和待办

### 痛点
手头事太多，总是忘记跟进，优先级一团乱。

### 具体操作步骤

**工具推荐：** Motion、Notion AI、Todoist AI

**用 AI 做日程管理的流程：**

1. 把所有待办事项丢给AI
2. 告诉它你的优先级规则——"紧急且重要"优先、"今天能搞定"优先
3. AI自动排好时间表，放到日历里
4. 每天自动生成"今日重点"卡片
5. 定期（每周末）让AI做复盘总结

**每日复盘指令：**
```
请根据我这周的完成情况，做以下复盘：
1. 本周完成了哪些关键成果（量化）
2. 哪些任务超时了？原因是什么？
3. 推荐下周优先级排序
4. 建议我砍掉哪些低价值事项
```
