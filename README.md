# 副业侦探 🕵️

聚合全网副业信息，帮你发现靠谱副业

## 功能
- 🏠 首页浏览最新副业推荐
- 📂 分类筛选（线上兼职/创业项目/骗局曝光）
- 🔍 搜索功能
- 📝 每日推荐
- 📖 博客文章

## 技术栈
- Next.js 14.2 (App Router)
- Tailwind CSS
- Vercel 部署
- Modern Minimal 设计风格

## 本地运行
```bash
npm install
npm run dev
```

## 部署
```bash
npm run build
vercel --prod
```

## 数据来源
贴吧、B站、知乎、小红书、虎扑
